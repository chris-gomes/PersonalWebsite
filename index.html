PROJECTS

Dangerous Links

The goal of this project was to develop and test machine learning models that could be used to determine if a site is malicious based off its URL and WHOIS information. Multiple machine learning models were trained and tested for accuracy and other metrics to determine the best model for different situations.

Media Stock Sentiment

The goal of this project was to determine if the performance of the stock market can be predicted from the public's response to events in the world. After collecting millions of articles and the public's response to these articles, multiple machine learning algorithms were trained and tested to try and determine future performance.

QwikTix.ai

The goal of this project was to design and create a database system for a movie information and ticketing service. The ddl SQL was created based on a relational database schema and ERD. In addition, a Python command line interface was written to perform basic operations and interactions with the database.
